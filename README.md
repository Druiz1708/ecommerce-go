# Sistema de Gestión de E-commerce

## Descripción
Este proyecto consiste en el desarrollo inicial de un sistema de gestión de e-commerce para una tienda de tecnología. El sistema busca organizar productos, controlar inventario, gestionar usuarios, registrar compras y coordinar envíos de manera más ordenada.

## Objetivo
Desarrollar una aplicación web básica en Go que permita planificar y estructurar los principales módulos de una tienda virtual, aplicando los conocimientos vistos en clase.

## Módulos principales
- Gestión de usuarios
- Catálogo de productos
- Inventario
- Gestión de compras
- Simulación de pagos
- Envíos

## Tecnologías utilizadas
- Go
- HTML
- CSS
- Templates HTML
- GitHub
